# Context Map

This repo is multi-context. Each context has its own CONTEXT.md glossary,
created lazily by /domain-modeling as domain terms get resolved.

| Context | Glossary         | Covers                                           |
| ------- | ---------------- | ------------------------------------------------ |
| infra   | infra/CONTEXT.md | Terraform, azapi, APIM, API Center (S3)          |
| src     | src/CONTEXT.md   | .NET Functions MCP server, APIM gateway (S1, S2) |

ADRs: system-wide decisions in docs/decisions/; context-scoped decisions may
live under infra/docs/decisions/ or src/docs/decisions/.
